Part I — Evaluate Limits
Problem 1 Easy
Evaluate using limit laws: $$\lim_{x \to 3} \left(2x^2 - 5x + 1\right)$$

Step-by-Step Solution

Step 1 — Direct Substitution
Since $f(x) = 2x^2 - 5x + 1$ is a polynomial, it is continuous everywhere. Apply direct substitution:
Step 2 — Evaluate
$$f(3) = 2(3)^2 - 5(3) + 1 = 18 - 15 + 1 = 4$$
$$\lim_{x \to 3} \left(2x^2 - 5x + 1\right) = \boxed{4}$$
Problem 2 Medium
Evaluate the limit using an analytic technique (factor and cancel): $$\lim_{x \to 2} \frac{x^2 - 4}{x - 2}$$
Try factoring the numerator as a difference of squares: $a^2 - b^2 = (a-b)(a+b)$

Step-by-Step Solution

Step 1 — Identify the Form
Direct substitution gives $\frac{0}{0}$, an indeterminate form. We need an analytic technique.
Step 2 — Factor
$$\frac{x^2 - 4}{x - 2} = \frac{(x-2)(x+2)}{x-2}$$
Step 3 — Cancel & Evaluate
For $x \neq 2$: $= x + 2$. So $\lim_{x\to 2}(x+2) = 4$.
$$\lim_{x \to 2} \frac{x^2 - 4}{x - 2} = \boxed{4}$$
Problem 3 Medium
Evaluate by rationalizing (conjugate technique): $$\lim_{x \to 0} \frac{\sqrt{x+4} - 2}{x}$$
Multiply top and bottom by the conjugate: $\sqrt{x+4} + 2$

Step-by-Step Solution

Step 1 — Multiply by Conjugate
$$\frac{\sqrt{x+4}-2}{x} \cdot \frac{\sqrt{x+4}+2}{\sqrt{x+4}+2} = \frac{(x+4)-4}{x(\sqrt{x+4}+2)} = \frac{x}{x(\sqrt{x+4}+2)}$$
Step 2 — Simplify & Evaluate
$$= \frac{1}{\sqrt{x+4}+2} \xrightarrow{x\to 0} \frac{1}{\sqrt{4}+2} = \frac{1}{4}$$
$$\lim_{x \to 0} \frac{\sqrt{x+4} - 2}{x} = \boxed{\frac{1}{4}}$$
Problem 4 Medium
Evaluate the limit using the Squeeze Theorem: $$\lim_{x \to 0} x^2 \cos\!\left(\frac{1}{x}\right)$$
Since $-1 \leq \cos(1/x) \leq 1$, what can you say about $-x^2 \leq x^2\cos(1/x) \leq x^2$?

Step-by-Step Solution

Step 1 — Set Up Bounds
We know $-1 \leq \cos(1/x) \leq 1$ for all $x \neq 0$. Multiply by $x^2 \geq 0$: $$-x^2 \leq x^2\cos\!\left(\frac{1}{x}\right) \leq x^2$$
Step 2 — Evaluate Bounding Limits
$$\lim_{x\to 0}(-x^2) = 0 \quad\text{and}\quad \lim_{x\to 0} x^2 = 0$$
Step 3 — Apply Squeeze Theorem
Since both bounds → 0, the function squeezed between them also → 0.
$$\lim_{x \to 0} x^2\cos\!\left(\frac{1}{x}\right) = \boxed{0}$$
Problem 5 Medium
Evaluate the limit at infinity: $$\lim_{x \to \infty} \frac{3x^3 - 2x + 1}{5x^3 + 4x^2 - 7}$$
Divide every term by the highest power in the denominator: $x^3$

Step-by-Step Solution

Step 1 — Divide by $x^3$
$$\frac{3x^3 - 2x + 1}{5x^3 + 4x^2 - 7} = \frac{3 - \frac{2}{x^2} + \frac{1}{x^3}}{5 + \frac{4}{x} - \frac{7}{x^3}}$$
Step 2 — Apply Limit
As $x \to \infty$, all $\frac{k}{x^n} \to 0$. So we get $\frac{3}{5}$.
$$\lim_{x \to \infty} \frac{3x^3 - 2x + 1}{5x^3 + 4x^2 - 7} = \boxed{\frac{3}{5}}$$
Problem 6 Hard
Determine the infinite limit and identify the vertical asymptote: $$\lim_{x \to 1^+} \frac{x + 3}{(x - 1)^2}$$

Step-by-Step Solution

Step 1 — Analyze the Sign
As $x \to 1^+$: the numerator $x+3 \to 4 > 0$. The denominator $(x-1)^2 \to 0^+$ (always positive since squared).
Step 2 — Conclude
$\frac{\text{positive}}{\text{small positive}} \to +\infty$. The line $x = 1$ is a vertical asymptote.
$$\lim_{x \to 1^+} \frac{x+3}{(x-1)^2} = \boxed{+\infty}$$
Vertical asymptote at $x = 1$.
Problem 7 Medium
Evaluate the special trig limit: $$\lim_{x \to 0} \frac{\sin(5x)}{3x}$$
Rewrite using the identity $\lim_{u\to 0}\frac{\sin u}{u} = 1$ by setting $u = 5x$.

Step-by-Step Solution

Step 1 — Rewrite
$$\frac{\sin(5x)}{3x} = \frac{5}{3}\cdot\frac{\sin(5x)}{5x}$$
Step 2 — Apply Known Limit
As $x \to 0$, $5x \to 0$ and $\frac{\sin(5x)}{5x} \to 1$. Therefore the limit is $\frac{5}{3}$.
$$\lim_{x \to 0}\frac{\sin(5x)}{3x} = \boxed{\frac{5}{3}}$$
Problem 8 Hard
Use the $\varepsilon$-$\delta$ definition of the limit to prove: $$\lim_{x \to 2} (3x - 1) = 5$$ Given $\varepsilon > 0$, find $\delta$ in terms of $\varepsilon$.

Step-by-Step Solution

Step 1 — Set Up
We need: $|f(x) - L| < \varepsilon$ whenever $0 < |x - a| < \delta$. $$|(3x - 1) - 5| < \varepsilon \implies |3x - 6| < \varepsilon \implies 3|x - 2| < \varepsilon$$
Step 2 — Solve for $\delta$
$$|x - 2| < \frac{\varepsilon}{3}$$ So choose $\delta = \frac{\varepsilon}{3}$.
Step 3 — Verify
If $0 < |x-2| < \delta = \varepsilon/3$, then $|(3x-1)-5| = 3|x-2| < 3\cdot\frac{\varepsilon}{3} = \varepsilon$. ✓
$$\delta = \boxed{\frac{\varepsilon}{3}}$$
Part II — Determine Continuity
Problem 9 Easy
State the three conditions required for $f$ to be continuous at $x = a$, and determine if the following function is continuous at $x = 1$: $$f(x) = \begin{cases} x^2 + 1 & x \neq 1 \\ 3 & x = 1 \end{cases}$$

Step-by-Step Solution

The Three Conditions
1. $f(a)$ is defined.
2. $\lim_{x\to a} f(x)$ exists.
3. $\lim_{x\to a} f(x) = f(a)$.
Check Each
✅ $f(1) = 3$ — defined.
✅ $\lim_{x\to 1} f(x) = \lim_{x\to 1}(x^2+1) = 2$ — exists.
❌ $\lim_{x\to 1} f(x) = 2 \neq 3 = f(1)$ — FAILS condition 3.
$f$ is discontinuous at $x = 1$ (removable discontinuity).
Problem 10 Medium
Determine where $g(x)$ is continuous: $$g(x) = \frac{x^2 - 9}{x^2 - 5x + 6}$$

Step-by-Step Solution

Step 1 — Factor the Denominator
$$x^2 - 5x + 6 = (x-2)(x-3)$$ Denominator is zero at $x = 2$ and $x = 3$.
Step 2 — Apply Continuity Rules
$g$ is a rational function. Rational functions are continuous everywhere they are defined, i.e., where the denominator $\neq 0$.
$g$ is continuous on $(-\infty, 2) \cup (2, 3) \cup (3, \infty)$.
Discontinuous at $\boxed{x = 2}$ and $\boxed{x = 3}$.
Problem 11 Medium
Find the value of $k$ that makes $f$ continuous at $x = 3$: $$f(x) = \begin{cases} kx + 1 & x \leq 3 \\ x^2 + 1 & x > 3 \end{cases}$$

Step-by-Step Solution

Step 1 — Left and Right Limits Must Match
Left: $\lim_{x\to 3^-} (kx+1) = 3k + 1$
Right: $\lim_{x\to 3^+} (x^2+1) = 10$
Step 2 — Solve
$$3k + 1 = 10 \implies 3k = 9 \implies k = 3$$
$$k = \boxed{3}$$
Problem 12 Medium
Use the Intermediate Value Theorem to show that $f(x) = x^3 + x - 1$ has a root in the interval $[0, 1]$.

Step-by-Step Solution

Step 1 — Verify Hypotheses
$f(x) = x^3 + x - 1$ is a polynomial, so it is continuous on $[0,1]$. ✅
Step 2 — Evaluate Endpoints
$f(0) = 0 + 0 - 1 = -1 < 0$
$f(1) = 1 + 1 - 1 = 1 > 0$
Step 3 — Apply IVT
Since $f$ is continuous on $[0,1]$ and $f(0) < 0 < f(1)$, by the IVT there exists $c \in (0,1)$ such that $f(c) = 0$. ∎
Problem 13 Easy
Which of the following functions are continuous for all real numbers? Select all that apply.
  • $f(x) = e^x$
  • $g(x) = \tan x$
  • $h(x) = x^4 - 3x^2 + 2$
  • $p(x) = \ln x$
  • $q(x) = \sin x$

Step-by-Step Solution

✅ (a) $e^x$ — exponential, continuous everywhere
❌ (b) $\tan x$ — undefined at $x = \frac{\pi}{2} + n\pi$
✅ (c) $x^4-3x^2+2$ — polynomial, continuous everywhere
❌ (d) $\ln x$ — only defined for $x > 0$
✅ (e) $\sin x$ — continuous everywhere
Continuous everywhere: $\boxed{(a),\ (c),\ (e)}$
Part III — Find & Interpret Derivatives
Problem 14 Medium
Use the limit definition of the derivative to find $f'(x)$ for: $$f(x) = 3x^2 - 2x + 5$$

Step-by-Step Solution

Step 1 — Set Up the Definition
$$f'(x) = \lim_{h\to 0}\frac{f(x+h) - f(x)}{h}$$
Step 2 — Expand $f(x+h)$
$$f(x+h) = 3(x+h)^2 - 2(x+h) + 5 = 3x^2 + 6xh + 3h^2 - 2x - 2h + 5$$
Step 3 — Compute Difference & Simplify
$$\frac{(3x^2+6xh+3h^2-2x-2h+5)-(3x^2-2x+5)}{h} = \frac{6xh+3h^2-2h}{h} = 6x+3h-2$$
Step 4 — Take Limit
$$\lim_{h\to 0}(6x+3h-2) = 6x - 2$$
$$f'(x) = \boxed{6x - 2}$$
Problem 15 Easy
A ball is thrown upward with position $s(t) = -16t^2 + 64t + 5$ (feet, seconds).
  • Find the velocity function $v(t) = s'(t)$.
  • Find the velocity at $t = 1$ second.
  • At what time does the ball reach its maximum height?

Step-by-Step Solution

(a) Velocity Function
$$v(t) = s'(t) = -32t + 64$$
(b) Velocity at $t = 1$
$$v(1) = -32(1) + 64 = 32 \text{ ft/s (upward)}$$
(c) Maximum Height
Set $v(t) = 0$: $-32t + 64 = 0 \implies t = 2$ seconds.
$v(t) = -32t+64$, $v(1) = 32$ ft/s, max height at $t = \boxed{2}$ sec.
Problem 16 Medium
Find the tangent line to $f(x) = x^3 - 4x$ at the point $(2, 0)$.

Step-by-Step Solution

Step 1 — Find $f'(x)$
$$f'(x) = 3x^2 - 4$$
Step 2 — Slope at $x=2$
$$f'(2) = 3(4) - 4 = 8$$
Step 3 — Point-Slope Form
$$y - 0 = 8(x - 2) \implies y = 8x - 16$$
$$y = \boxed{8x - 16}$$
Problem 17 Medium
Differentiate using the product and chain rules: $$y = e^{2x}\sin(3x)$$

Step-by-Step Solution

Step 1 — Product Rule
Let $u = e^{2x}$ and $v = \sin(3x)$. Then $y' = u'v + uv'$.
Step 2 — Chain Rule for Each
$u' = 2e^{2x}$ (chain rule: derivative of $2x$ is 2)
$v' = 3\cos(3x)$ (chain rule: derivative of $3x$ is 3)
Step 3 — Combine
$$y' = 2e^{2x}\sin(3x) + 3e^{2x}\cos(3x) = e^{2x}\left[2\sin(3x) + 3\cos(3x)\right]$$
$$y' = \boxed{e^{2x}\left[2\sin(3x) + 3\cos(3x)\right]}$$
Problem 18 Medium
Differentiate using the quotient rule: $$h(x) = \frac{\cos x}{1 + \sin x}$$

Step-by-Step Solution

Step 1 — Quotient Rule
$$h'(x) = \frac{(-\sin x)(1+\sin x) - (\cos x)(\cos x)}{(1+\sin x)^2}$$
Step 2 — Expand Numerator
$$= \frac{-\sin x - \sin^2 x - \cos^2 x}{(1+\sin x)^2} = \frac{-\sin x - (\sin^2 x + \cos^2 x)}{(1+\sin x)^2}$$
Step 3 — Pythagorean Identity
$$= \frac{-\sin x - 1}{(1+\sin x)^2} = \frac{-(1+\sin x)}{(1+\sin x)^2} = \frac{-1}{1+\sin x}$$
$$h'(x) = \boxed{\frac{-1}{1 + \sin x}}$$
Problem 19 Hard
Find the second derivative $f''(x)$ of: $$f(x) = \ln(x^2 + 1)$$

Step-by-Step Solution

Step 1 — First Derivative (Chain Rule)
$$f'(x) = \frac{1}{x^2+1}\cdot 2x = \frac{2x}{x^2+1}$$
Step 2 — Second Derivative (Quotient Rule)
$$f''(x) = \frac{2(x^2+1) - 2x(2x)}{(x^2+1)^2} = \frac{2x^2 + 2 - 4x^2}{(x^2+1)^2} = \frac{2 - 2x^2}{(x^2+1)^2}$$
$$f''(x) = \boxed{\frac{2(1-x^2)}{(x^2+1)^2}}$$
Problem 20 Hard
A function $f$ has the graph shown below. At which points is $f$ not differentiable? Classify each as: corner, cusp, vertical tangent, or discontinuity.

Step-by-Step Solution

$f$ is not differentiable at:

$x = -2$: Jump discontinuity — left and right limits don't match, so $f$ is not continuous (and differentiability requires continuity).

$x = 0$: Corner — the function is continuous but the left derivative $\neq$ right derivative (the slope changes abruptly).

$x = 2$: Cusp — the left and right derivatives approach $+\infty$ and $-\infty$ respectively.
Not differentiable at $x = -2$ (discontinuity), $x = 0$ (corner), $x = 2$ (cusp).
